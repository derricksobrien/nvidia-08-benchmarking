# Benchmarking & Latency Optimization

> **Track:** NVIDIA HPC & Developer Upskilling (Track 2)
> **Course 8 of 8**

Profile, instrument, and optimize compute workloads to hit production latency targets on NVIDIA hardware.

---

## Modules

### Module 1: Profiling Tools

**Learning Objectives:**
- Profile with Nsight Systems
- Identify CPU-GPU sync bottlenecks
- Read timeline views and traces

### Module 2: Memory Bandwidth Optimization

**Learning Objectives:**
- Calculate roofline model limits
- Reduce unnecessary memory copies
- Apply memory access coalescing

### Module 3: Kernel Fusion Techniques

**Learning Objectives:**
- Identify fusable kernel pairs
- Implement custom fused CUDA kernels
- Measure reduction in kernel launch overhead

### Module 4: Production Latency Targets

**Learning Objectives:**
- Define SLA budgets per inference stage
- Implement adaptive batching
- Validate sub-500ms end-to-end latency
